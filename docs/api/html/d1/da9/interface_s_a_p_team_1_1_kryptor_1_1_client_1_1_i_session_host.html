Handles requests from the sessions.

This method should send requests to user and if it's not possible, it could return the SessionRequest<TResponse>.DefaultValue or throws an exception. The thrown exception would be catched on the sender session and the session will fail.

Parameters

requestThe request data.

Returns
The session host or user response.

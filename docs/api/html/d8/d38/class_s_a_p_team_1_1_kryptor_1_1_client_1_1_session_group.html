◆ EndReason

SessionEndReason SAPTeam.Kryptor.Client.SessionGroup.EndReason
get

Gets a brief report of all session's end reasons.

When all sessions ended successfully, the end reason would be SessionEndReason.Completed.

When there is any cancelled session, the end reason would be SessionEndReason.Cancelled.

When there is any failed session, the end reason would be SessionEndReason.Failed.

When there is any skipped session, the end reason would be SessionEndReason.Skipped.

When there is any wierd session or when the session group is not started or still running with no ended session, the end reason would be SessionEndReason.None.

Except for the first case, in other cases the first occurrence is considered. even when there are still running or not started sessions.

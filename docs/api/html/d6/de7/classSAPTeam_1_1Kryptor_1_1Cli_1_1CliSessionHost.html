override async Task< TResponse > SAPTeam.Kryptor.Cli.CliSessionHost.OnSessionRequest< TResponse > ( ISession session,
SessionRequest< TResponse > request,
CancellationToken cancellationToken )

Handles requests from the sessions.

This method should send requests to user and if it's not possible, it could return the SessionRequest<TResponse>.DefaultValue or throws an exception. The thrown exception would be catched on the sender session and the session will fail.

Parameters
sessionThe paused session.
requestThe request data.
cancellationTokenA cancellation token to monitor the task.
Template Parameters
TResponseThe type of requested response from user.
Returns
The session host or user response.
Exceptions
NotSupportedExceptionThis exception thrown when the requested TResponse is not supported by this session host.

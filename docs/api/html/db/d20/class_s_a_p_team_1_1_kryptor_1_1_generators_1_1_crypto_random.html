Detailed Description

A random number generator based on the RNGCryptoServiceProvider. Adapted from the "Tales from the CryptoRandom" article in MSDN Magazine (September 2007) but with explicit guarantee to be thread safe. Note that this implementation also includes an optional (enabled by default) random buffer which provides a significant speed boost as it greatly reduces the amount of calls into unmanaged land.
